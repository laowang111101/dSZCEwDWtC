# 教务管理系统简介 python649

## 项目概述

本教务管理系统基于 Python Django 框架，采用前后端分离的技术架构开发。系统致力于提供一套完整、高效的教务管理解决方案，适用于各类学校和教育机构。

## 技术栈

- 后端：Python Django
- 数据库：MySQL

## 功能模块

### 用户角色

系统定义了以下三种角色，每种角色拥有不同的功能页面：

- 管理人员
- 教师
- 学生

### 功能概览

- 登录/注册功能
- 管理人员界面：全面管理教务信息
- 教师界面：处理选课、成绩录入等教学事务
- 学生界面：选课、查询成绩等
- 可视化增删改查操作

## 系统角色

- 系统管理员：负责维护系统运行，管理用户权限等
- 教师：负责课程教学，成绩录入等
- 学生：进行选课，查看成绩等

## 运行环境

- 后端：Python 环境推荐使用 Python 3.x
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 安装 Python 环境及相关依赖
2. 配置 MySQL 数据库
3. 部署后端服务
4. 前端页面部署（假设有独立的前端项目）
